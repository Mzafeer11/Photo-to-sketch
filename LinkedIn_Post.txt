✏️ Bridging the Domain Gap: Photo-to-Sketch with CycleGAN

How do you translate images between domains when you don't have paired data? My latest project implements CycleGAN on the "Sketchy" dataset to automate realistic sketch generation from photos.

Key Engineering Highlights:
🔹 Cycle-Consistency Loss: Enforced that translating a photo to a sketch and back result in the original image, preventing mode collapse in unpaired sets.
🔹 ResNet-9 Architecture: Utilized a deep 9-block ResNet Generator to handle complex style transformations while preserving global object geometry.
🔹 Identity Mapping: Integrated an Identity Loss term to maintain consistency when the input image already shares characteristics with the target domain.

The repository includes a fully modularized training script with automated checkpointing and mixed-precision support.
